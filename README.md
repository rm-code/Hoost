## Controls

| Key       | Action                     |
|-----------|----------------------------|
| Left      | Select previous fire mode  |
| Right     | Select next fire mode      |
| s         | Stand                      |
| c         | Crouch                     |
| p         | Prone                      |
| r         | Reload Weapon              |
| a         | Switch to Attack Mode      |
| m         | Switch to Movement Mode    |
| e         | Switch to Interaction Mode |
| i         | Open Inventory             |
| Backspace | Select previous Character  |
| Space     | Select next Character      |
| Return    | End Turn                   |
